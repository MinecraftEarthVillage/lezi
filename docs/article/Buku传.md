# 继续清实体

使用插件删除生物是笨蛋中的笨蛋才会做的事，生物如果达到服务器设定的上限则会停止生成. 而被清除后，服务器必须重新生成生物，这个过程也是非常费性能的。

如果你不需要那么多生物，直接调整服务端配置文件参数参数让他不生成那么多即可
